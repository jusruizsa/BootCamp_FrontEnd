Para el desarrollo de esta actividad se pidió estudiar etiquetas de HTML y comprender cuáles de estas se usan con mayor frecuencia en proyectos reales, también se pidió aplicar estilos CSS de forma libre. Esta página es el resultado de esta actividad.

Etiquetas utilizadas en el desarrollo de esta actividad

<main>: Representa el contenido principal del <body>.

<p>: Etiqueta utilizada para la construcción de párrafos.

<header>: Utilizado principalmente para la navegación, pero puede contener cualquier tipo de información, como encabezados, logos, información de autor, etc.

<h1> <h2> <h3> <h5>: Utilizado en encabezados (títulos) van desde h1 el más importante hasta h6 el menos importante.

<section>: Se utiliza para encerrar una sección del documento es similar a <div> la diferencia es que este último no tiene una intención semántica.

<code>: Sirve para insertar texto con formato de código de programación.

<strong>: Utilizado para enfatizar texto, el texto entre estas etiquetas se muestra en negrilla, visualmente tiene el mismo efecto que <b>.

<en>: Utilizado para enfatizar texto, el texto entre estas etiquetas se muestra en letra itálica, visualmente tiene el mismo efecto que <i>.

<img>: Utilizado para insertar imágenes en el documento.

<footer>: Representa el pie de pagina y suele ser utilizado para información de contacto, redes sociales, etc.

<a>: Utilizado para enlaces ya sean externos o de la página propia.
